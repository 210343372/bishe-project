📐 全球光伏资源评估系统 - 计算原理

1. 光伏发电量仿真模型

本系统基于经纬度、真实海拔数据，结合气象学经验模型计算光伏发电量，核心逻辑如下：

1.1 太阳总辐射量 (GHI) 计算

基于纬度和海拔修正基础辐射量（纬度越高辐射越低，海拔越高辐射越高）：

基础GHI = 1800 - (|纬度| / 90) × 800
海拔修正 = (海拔 / 1000) × 0.05 × 基础GHI
年总GHI = 基础GHI + 海拔修正

其中，海拔修正系数取0.05/1000m（即海拔每升高1000米，辐射量增加5%）。

1.2 组件温度与温度损失

光伏组件温度随环境温度和辐射量变化，进而影响发电效率：

组件温度 = 环境温度 + (NOCT - 20) × (GHI / 800)
温度损失系数 = 1 - 0.004 × (组件温度 - 25)

其中，NOCT（标称工作电池温度）取45℃，温度损失系数为-0.4%/℃（行业标准值）。

1.3 月度发电量计算

月度发电量 (kWh) = 月度GHI × 1.1 × 组件效率 × 性能比率 × 温度损失系数 × 系统容量 / 1000

关键参数取值：组件效率 η=0.18（18%），系统性能比率 PR=0.80（80%），斜面辐射修正系数 1.1。

2. 光伏项目经济性评估模型

基于全生命周期成本（LCOE）、投资回收期、内部收益率（IRR）评估项目可行性：

2.1 初始投资与年收益

初始投资 (万元) = 系统容量 (kW) × 单位投资 (3500元/kW) / 10000
年发电收益 (万元) = 年发电量 (kWh) × 上网电价 (0.55元/kWh) / 10000
年运维成本 (万元) = 初始投资 × 运维费率 (1.5%)

2.2 核心评估指标

1. 投资回收期 (年) = 初始投资 / (年收益 - 年运维成本)
2. 度电成本 (LCOE, 元/kWh) = 全生命周期总成本 / 全生命周期总发电量
3. 内部收益率 (IRR, %) = (年净收益 / 初始投资) × 100

其中，项目寿命按25年计算，组件效率年衰减率为0.5%。

3. 数据来源

  • 地图服务：高德地图JS API 2.0
  • 真实海拔数据：Open-Elevation 免费API（基于SRTM90全球地形数据，90米分辨率）
  • 气象模型：基于纬度-海拔的经验辐射模型（参考光伏行业通用设计规范）
  • 经济参数：参考国内2025-2026年光伏电站平均建设与运维成本
